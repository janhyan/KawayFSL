Why Kaway?

A shortage of proficient special education teachers and accessible learning materials hinders learners from effectively learning FSL.

Current research on FSL recognition is limited to basic detection, often invasive, and lacks comprehensive systems that provide feedback to users. Kaway is an automatic FSL tutor that uses image processing and deep learning to recognize both static and dynamic gestures. Accessible via desktop and web applications, the system enables students to take interactive modules while providing real-time feedback on hand gestures using 42 static hand feature landmarks and 1,662 dynamic key points for hand, pose, and facial features through Long Short-Term Memory (LSTM) networks. Kaway-FSL focused on 50 essential FSL gestures based on the Kindergarten curriculum, covering alphabet knowledge, vocabulary development, and proper expressions for introducing oneself and polite greetings.
